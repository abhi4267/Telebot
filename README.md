# Weather-bot

## About the Project
It is a web application made using React and NestJS to know the weather of any city. It fetches data using OpenWeather OneCall API. It has great UI and features

## Technologies Used
* React
* OpenWeather API
* NodeJS
* Tailwind
* NestJS
